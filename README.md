# Book Notes Project

A full-stack book notes application built with Node.js, Express, PostgreSQL, and EJS.

## Features

- Add books
- Edit books
- Delete books
- Store book reviews and ratings
- Display book covers using ISBN
- Sort books by rating, title, or date

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- EJS
- dotenv
- body-parser
- axios
